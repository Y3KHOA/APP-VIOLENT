* ỨNG DỤNG PYTHON
-Bước 1: Vào thư mục Lib/Python/pytorch
-Bước 2: thiết lập thư viện trong IDE python (ở đây chúng tôi dùng pycharm)
Vào File > Settings > Project > Python Interpreter
-Bước 3: add thư viện có đường dẫn trên vào python
Python Interpreter > Add Python Interpreter 
-Bước 4: add thư viện bằng conda
Conda Evironment > Location (trỏ địa chỉ chứa file Lib vào)
-Bước 5: chạy LearnGUI_V2.py
